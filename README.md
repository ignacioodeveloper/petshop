## Clonar Projecto Localmente

Clonar Projecto

```bash
  git clone [https://github.com/FJosephx/MascotasFelices]
```

Ir a la Carpeta

```bash
  cd my-project
```

Activar Ambiente Virtual

```bash
  scripts activate.bat
```

Inicia Django

```bash
  python manage.py runserver
```
